selector

selector is used to select the elements in the html

selector is used to apply css to selected the elements in the html

selector is used to apply js to selected the elements in the html

types of selector

  1. element selector
  2. id selector
  3. class selector
  4. attribute
  5. universal
  6. group
  7. descendant
  8. child
  9. adjacent
  10. general

element selector

element selector is used to select the elements based on tag name

class selector

  • class selectoris used to select the elements based on the class
  • class selector not to be unique
  • we can use class selector with .className for css style

id selector

  • id selector is used to select elements based on id name
  • id selector must be unique
  • it is used to select based on id name

Rules:

  • class/id name should have alphabets and numbers(-,_)
  • class/id should not start with number
  • class name and id name can be same
